    5.快速利用hexo搭建博客

    Hexo和GitHub搭建博客的原理是：Hexo将source下的md文件生成静态的html页面，存放到public目录中，这一步是由命令：hexo -g完成。接下来执行hexo -d命令，就将public目录下的文件推送到了github上。github上显示的就是这些静态页面。
