从现在开始要开始搞论文啦，2020.12.28第一次打卡

题目：基于node.js的知识社区实现与管理

在一开始准备论文的时候并不确定要做什么，而且当时正在准备考研，突然被通知要选导师了！！！

ps：就在前几个小时前刚跟我闺蜜嘚瑟过我现在不着急论文，划掉划掉

幸好的是之前在准备考研的时候，有点儿烦躁，就去搜了搜怎么美化个人博客，本身也挺感兴趣的，17年的时候注册了第一个博客，那时候跟风嘛，就没事发发代码、笔记什么的，后来嫌弃页面丑，就又注册了博客园（因为可以自己搞好看的页面）然后还是不满足吧，在搜索问题的时候，看到各位大佬们自己搭建的个人博客，好好看！！！就没事的时候搜一搜，倒是没想到算是为论文准备打了个铺垫吧。

个人博客搭建现在热门的有两种wordpress和hexo

  • hexo是一款基于Node.js的静态博客框架，可以很方便的生成静态网页托管在GitHub上，相较于其他博客框架来说，hexo的编译速度更快，搭建步骤也更加简单，中文支持也更好。
  • 考虑到后期我需要修改页面，而WordPress是基于PHP开发的，本人并不会，而且WordPress相对于hexo比较臃肿并不适合我的选择，于是我就选择了基于node.js的hexo博客系统 : )

编辑器Visual Studio Code

我选的这个版本，操作起来相对容易(主要是纯黑的编辑页面I like)
